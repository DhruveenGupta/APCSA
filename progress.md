2️⃣ How to use progress.md

Daily Updates:

After completing notes + practice + FRQ + mini-project, mark the day as ✅.

Update concept mastery percentages based on how comfortable you feel.

FRQs:

After solving an FRQ, mark the box with ✅ or partially filled (⬛) if partially done.

Mini Projects:

Mark as ✅ when fully implemented, compiled, and tested.

Weekly Reviews:

On Sunday (end of the week), update all concept percentages and FRQ completion to see progress.

3️⃣ Optional Enhancements

Add color-coded emojis:

✅ = done, ⚠ = needs revision, ⬛ = partially done

Add notes/comments column for tricky concepts or mistakes.
